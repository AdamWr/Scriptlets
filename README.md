# AdGuard Scriptlets

Install dependencies
```
yarn install
```

Build for Extension
```
yarn build
```

Build for Corelibs
```
yarn corelibs
```

Build dev (rebuild js files on every change)
```
yarn watch
```

Run UI Unit testing
```
yarn test
```

---

Please sure that all scriptlets compiled in `dist/build.js` have no outer dependencies generated by `Babel`
